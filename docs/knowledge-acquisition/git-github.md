# Git and GitHub

## Why It Matters

Tex Wiki is developed in a public GitHub repository and uses Git to track product and documentation evolution.

## Topics To Learn

- Repositories
- Commits
- Branches
- Remotes
- Pull requests
- Issues
- Releases
- Topics

## Project Applications

- Version the extension source code.
- Publish documentation.
- Track roadmap items.
- Prepare Marketplace releases.
