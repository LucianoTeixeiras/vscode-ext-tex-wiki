# Marketplace Publishing

## Why It Matters

Publishing to the Visual Studio Marketplace makes Tex Wiki installable by other VS Code users.

## Topics To Learn

- VS Code publisher account
- Personal Access Token
- `@vscode/vsce`
- `.vsix` package
- Extension metadata
- Versioning
- Release notes

## Project Applications

- Package Tex Wiki.
- Test local `.vsix` installation.
- Publish the extension.
- Maintain release documentation.

## Practice Tasks

1. Install `@vscode/vsce`.
2. Run `vsce package`.
3. Inspect package contents before publishing.
