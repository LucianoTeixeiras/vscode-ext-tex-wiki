# Node.js and npm

## Why It Matters

VS Code extensions run on Node.js inside the extension host.

## Topics To Learn

- `package.json`
- npm scripts
- dependencies and devDependencies
- Node.js modules
- semantic versioning

## Project Applications

- Compile TypeScript.
- Manage extension dependencies.
- Package the extension.
- Run development scripts.

## Practice Tasks

1. Understand every field in `package.json`.
2. Run `npm run compile`.
3. Add future scripts for linting and testing.
