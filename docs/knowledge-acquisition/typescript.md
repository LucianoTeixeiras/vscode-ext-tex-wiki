# TypeScript

## Why It Matters

Tex Wiki is written in TypeScript because the VS Code Extension API is strongly typed and the extension will benefit from explicit models for scan results, wiki pages, and generated diagrams.

## Topics To Learn

- Basic types
- Interfaces
- Type aliases
- Functions
- Async and promises
- Modules and imports
- Strict mode
- Error handling

## Project Applications

- Model workspace scan results.
- Model generated wiki pages.
- Type renderer functions.
- Prevent invalid wiki structures.

## Practice Tasks

1. Create a `WikiPage` interface.
2. Create a `WorkspaceScanResult` interface.
3. Create typed functions for Markdown generation.
