# VS Code Extension API

## Why It Matters

Tex Wiki depends on the VS Code Extension API to register commands, read the active workspace, write files, and interact with the editor.

## Topics To Learn

- Extension activation
- Commands
- Workspace folders
- `vscode.workspace.fs`
- Notifications
- Output channels
- Configuration

## Project Applications

- Register `Tex Wiki: Generate Wiki`.
- Detect the current workspace.
- Create the generated wiki folder.
- Write Markdown files.
- Open generated files in the editor.

## Practice Tasks

1. Add an output channel.
2. Add a progress notification during generation.
3. Add configuration for output folder.
