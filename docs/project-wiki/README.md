## Product Purpose

Tex Wiki generates rich project documentation from the folder currently open in VS Code.

The generated wiki should include:

- Project overview
- Directory map
- Architecture notes
- Main flows
- Setup instructions
- Deployment notes
- Mermaid diagrams
- Glossary
